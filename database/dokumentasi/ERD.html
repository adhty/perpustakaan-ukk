🔑 Keterangan Simbol

PK = Primary Key
FK = Foreign Key
UK = Unique Key
NULL = Opsional

Relasi antar tabel:

• users ←→ pinjams (1:N)

• bukus ←→ pinjams (1:N)

• kategoris ←→ bukus (1:N)

• users (admin) ←→ pinjams (1:N via admin_id)

🔗 Relasi Antar Tabel

Tabel Asal Kardinalitas Tabel Tujuan Foreign Key Aturan Hapus Keterangan
