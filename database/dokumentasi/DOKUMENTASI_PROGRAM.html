📋 I. DESKRIPSI PROGRAM

1.1 Gambaran Umum

Aplikasi Perpustakaan Digital Sekolah adalah sistem manajemen peminjaman buku berbasis web yang dikembangkan menggunakan framework Laravel 11 (PHP) dengan database MySQL. Aplikasi ini berjalan secara lokal (localhost) tanpa memerlukan koneksi internet, dan dirancang untuk memudahkan proses peminjaman dan pendataan buku di lingkungan sekolah.

2.6 Middleware – RoleMiddleware

Fungsi: Membatasi akses halaman berdasarkan role pengguna (admin/siswa).

Cara kerja:

  1. Cek apakah user sudah login (Auth::check()). Jika belum, redirect ke login.
  2. Bandingkan Auth::user()->role dengan parameter role yang diizinkan.
  3. Jika tidak sesuai, lempar HTTP 403 Forbidden.
  4. Jika sesuai, lanjutkan ke handler berikutnya ($next($request)).

Penggunaan: Route::middleware(['auth', 'role:admin']) atau role:siswa

4.2 Evaluasi Sistem

✅ Kelebihan Sistem:
  • Antarmuka modern, responsif, dan mudah digunakan
  • Validasi data yang komprehensif di sisi server (server-side validation)
  • Sistem denda otomatis mengurangi pekerjaan manual petugas
  • Pemisahan hak akses admin dan siswa yang ketat melalui middleware
  • Pencarian real-time dengan multiple field (judul, pengarang, kode, ISBN)
  • Grafik peminjaman memudahkan monitoring tren penggunaan perpustakaan
  • Fitur cetak laporan langsung dari browser tanpa plugin tambahan
⚠️ Keterbatasan & Saran Pengembangan:
  • Belum ada fitur notifikasi email / SMS untuk pengingat batas kembali
  • Belum ada fitur barcode scanner untuk input kode buku secara cepat
  • Laporan belum dapat diekspor ke format Excel/PDF secara langsung
  • Belum ada fitur reservasi / antrian buku yang sedang habis
  • Rekomendasi: tambahkan fitur backup database otomatis berkala

4.3 Kesimpulan

Aplikasi Perpustakaan Digital Sekolah berhasil dikembangkan menggunakan framework Laravel dengan memenuhi seluruh kebutuhan fungsional yang ditetapkan dalam soal UKK RPL 2025/2026. Sistem ini mampu melayani dua jenis pengguna (admin dan siswa) dengan hak akses yang berbeda, menyediakan fitur CRUD lengkap untuk pengelolaan data buku, anggota, dan transaksi peminjaman, serta dilengkapi dengan penghitungan denda otomatis dan pelaporan berbasis waktu.
